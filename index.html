CCNA 1

May 2024

The CCNA 1 (Cisco Certified Network Associate) certification is one of the first steps in the Cisco certification path for those who want to learn computer network skills. The CCNA certification covers a wide range of network concepts, but the "CCNA 1" part refers to the first course of the program, called Introduction to Networks (ITN).

Main topics of CCNA 1:

  1. Network Fundamentals: Introduction to basic network concepts, like network parts (routers, switches, access points) and how they talk to each other.
  2. OSI and TCP/IP Models: Deep dive into communication models between network devices, focusing on how data moves between the layers of the OSI (Open Systems Interconnection) and TCP/IP models.
  3. IP Addressing: Introduction to IPv4 and IPv6 addresses, their structure, and how they are used to identify devices on a network. It also covers subnet calculation and creating IP addressing plans.
  4. Ethernet and Switching: Concepts related to Ethernet technology and packet switching inside a LAN. It covers MAC addresses, switching tables, and collision domains.
  5. Basic Router and Switch Configuration: Students learn to set up routers and switches using the command line interface (CLI). This includes setting up network interfaces, routing protocols, and other basic functions.
  6. Network Protocols: Study of main network protocols, like ARP (Address Resolution Protocol), DHCP (Dynamic Host Configuration Protocol), and ICMP (Internet Control Message Protocol), and how they work in moving data between network devices.
  7. Network Security: Introduction to basic network security concepts, like using passwords for network devices and setting up access control lists (ACL) to limit unauthorized traffic.

Learning Goals

  • Understand the basic concepts of computer networks.
  • Know how to configure and fix basic problems on LAN and WAN networks.
  • Gain skills in using the command interface to configure network devices.
  • Understand how IP addresses, network protocols, and data switching work.

IT ESSENTIALS

May 2024

The IT Essentials course, offered by Cisco, is designed to provide basic skills in computer science and information technology. The program focuses on basic concepts of computers, hardware, software, security, and troubleshooting, giving the foundation for an IT career.

Main topics of IT Essentials:

  1. Hardware Components: Description of the main computer parts, like CPU, RAM, hard disk, power supplies, motherboards, and peripherals. Students learn how to build, take apart, and identify PC hardware parts.
  2. Operating Systems: Installation and management of main operating systems, like Microsoft Windows, Linux, and macOS. It covers basic setup, updates, and managing system settings.
  3. Computer Networks: Introduction to computer networks and basic concepts, like LAN, WAN, network protocols, network devices, and IP addressing. Students learn to set up and fix basic network problems.
  4. Cybersecurity: Key security concepts, including data protection, access management, and common threats like malware and phishing. Students learn to use basic security measures.
  5. Maintenance and Troubleshooting: Troubleshooting techniques to find and fix hardware and software problems. It covers both common problems and ways to prevent future breakdowns.
  6. Mobility and Devices: Management of mobile devices like smartphones and tablets, including setting up network settings and syncing data between mobile devices and PCs.
  7. Cloud Services and Virtualization: Introduction to cloud services and virtualization. Students learn how to set up virtual machines and understand basic concepts of cloud computing.

Learning Goals

  • Understand the hardware components of a computer and their functions.
  • Install, configure, and update operating systems.
  • Gain basic skills in configuring networks and fixing network problems.
  • Use cybersecurity measures and recognize common threats.
  • Fix hardware and software problems.
  • Manage and configure mobile devices and understand virtualization technologies.
